       - `GetProductByID`: SELECT product by id. Return `ErrProductNotFound` on `sql.ErrNoRows`.
       - `ListIndicationCodes`: SELECT from indication_codes WHERE active=true. If moduleType provided, add AND module_type=$1. Order by module_type, name.
       - `ListClinicalEndpoints`: SELECT from clinical_endpoints WHERE active=true. If moduleType provided, add AND module_type=$1. Order by module_type, name.
