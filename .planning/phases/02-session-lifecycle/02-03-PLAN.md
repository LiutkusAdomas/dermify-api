  - Struct: db *sql.DB. Constructor: NewPostgresConsentRepository(db *sql.DB).
  - Create: INSERT INTO session_consents with all fields, RETURNING id.
  - GetBySessionID: SELECT by session_id (UNIQUE constraint means at most one row). Return ErrConsentNotFound via sql.ErrNoRows wrapping.
  - Update: UPDATE session_consents SET ... WHERE id = $1 AND version = $expected. Increment version. Zero rows = version conflict error.
  - ExistsForSession: SELECT EXISTS(SELECT 1 FROM session_consents WHERE session_id = $1). Returns bool.

  **PostgresContraindicationRepository (contraindication.go):**
  - Struct: db *sql.DB. Constructor: NewPostgresContraindicationRepository(db *sql.DB).
  - Create: INSERT INTO contraindication_screenings with all boolean flag columns, metadata. RETURNING id.
  - GetBySessionID: SELECT by session_id (UNIQUE constraint). Return ErrScreeningNotFound via sql.ErrNoRows wrapping.
  - Update: UPDATE contraindication_screenings SET ... WHERE id = $1 AND version = $expected. Increment version.

  **PostgresModuleRepository (session_module.go):**
  - Struct: db *sql.DB. Constructor: NewPostgresModuleRepository(db *sql.DB).

  - NextSortOrder: SELECT COALESCE(MAX(sort_order), 0) + 1 FROM session_modules WHERE session_id = $1.
