  **internal/api/handlers/sessions.go -- ALL handlers follow closure pattern:**
  Each handler is a function returning `func(http.ResponseWriter, *http.Request)`.

  - **HandleCreateSession(svc *service.SessionService, m *metrics.Client):** Parse createSessionRequest (patient_id required, optional header fields). Build domain.Session, set ClinicianID from JWT claims. Call svc.Create. Map errors: ErrInvalidSessionData->400, ErrPatientNotFound->404, default->500. On success: 201 + SessionResponse. Increment session created counter.
  - **HandleGetSession(svc, m):** Parse {id} from URL. Call svc.GetByID. Map ErrSessionNotFound->404. Return SessionResponse.
  - **HandleListSessions(svc, m):** Parse query params (patient_id, clinician_id, status, page, per_page). Call svc.List. Return PaginatedResponse with sessions.
  - **HandleUpdateSession(svc, m):** Parse {id} and request body. Build domain.Session with ID, Version from request. Set UpdatedBy from claims. Call svc.Update. Map errors: ErrSessionNotEditable->409, ErrSessionVersionConflict->409, ErrInvalidSessionData->400, ErrSessionNotFound->404. Return updated SessionResponse.
  - **HandleTransitionSession(svc, m):** Parse {id} and request body ({status: "in_progress"}). Call svc.TransitionState with userID from claims. Map ErrInvalidStateTransition->409, ErrSessionNotFound->404, ErrSessionVersionConflict->409. Return 200 with {message: "state transitioned"}.

  - **HandleRecordConsent(svc *service.ConsentService, m):** Parse {id} (session_id) and request body. Build domain.Consent. Call svc.RecordConsent. Map errors: ErrConsentAlreadyExists->409, ErrInvalidConsentData->400. Return 201 + ConsentResponse.
  - **HandleGetConsent(svc, m):** Parse {id}. Call svc.GetBySessionID. Map ErrConsentNotFound->404. Return ConsentResponse.
  - **HandleUpdateConsent(svc, m):** Parse {id} and body. Call svc.UpdateConsent. Map errors appropriately. Return updated ConsentResponse.

  - **HandleGetScreening(svc, m):** Parse {id}. Call svc.GetBySessionID. Map ErrScreeningNotFound->404. Return ScreeningResponse.
