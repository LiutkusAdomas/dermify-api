Create internal/api/handlers/photo.go with:
1. HandleUploadBeforePhoto(svc *service.PhotoService, m *metrics.Client) func(http.ResponseWriter, *http.Request):
   - Set Content-Type: application/json
   - Extract claims via middleware.GetUserClaims (401 if nil)
   - Parse session ID from chi.URLParam(r, "id") (400 if invalid)
   - Wrap r.Body with http.MaxBytesReader(w, r.Body, domain.MaxPhotoSize)
   - Parse file via r.FormFile("photo") (400 if error)
   - defer file.Close()
   - Read first 512 bytes for content type detection via http.DetectContentType
   - IMPORTANT: file.Seek(0, io.SeekStart) after detection to reset reader
   - Build domain.Photo with SessionID, PhotoType=domain.PhotoTypeBefore, OriginalName=header.Filename, ContentType=detected type, SizeBytes=header.Size, CreatedBy=claims.UserID, UpdatedBy=claims.UserID
   - Call svc.UploadPhoto(r.Context(), photo, file)
   - On error: handlePhotoError
   - On success: m.IncrementPhotoUploadedCount(), w.WriteHeader(201), encode photo as JSON

2. HandleUploadLabelPhoto(svc *service.PhotoService, m *metrics.Client) func(http.ResponseWriter, *http.Request):
   - Same as before photo but:
   - Also parse moduleId from chi.URLParam(r, "moduleId") (400 if invalid)
   - Set PhotoType=domain.PhotoTypeLabel and ModuleID=&moduleID

3. HandleListSessionPhotos(svc *service.PhotoService, m *metrics.Client) func(http.ResponseWriter, *http.Request):
   - Parse session ID from URL
   - Call svc.ListBySession
   - Return 200 with JSON array

4. HandleGetPhoto(svc *service.PhotoService, m *metrics.Client) func(http.ResponseWriter, *http.Request):
   - Parse photoId from chi.URLParam(r, "photoId")
   - Call svc.GetByID
   - Return 200 with JSON photo metadata

5. HandleServePhotoFile(svc *service.PhotoService, basePath string) func(http.ResponseWriter, *http.Request):
   - Parse photoId from URL
   - Call svc.GetByID to get photo metadata (path is server-controlled, not user-supplied -- prevents path traversal)
   - Build full path: filepath.Join(basePath, filepath.FromSlash(photo.FilePath))
   - Set Content-Type header from photo.ContentType
   - Use http.ServeFile(w, r, fullPath)

6. HandleDeletePhoto(svc *service.PhotoService, m *metrics.Client) func(http.ResponseWriter, *http.Request):
   - Parse session ID and photoId from URL
   - Extract claims for userID
   - Call svc.DeletePhoto(ctx, photoID, claims.UserID)
   - Return 204 No Content on success

Use parseIDParam pattern from existing handlers for session ID. Create a local parsePhotoIDParam helper or use strconv.ParseInt(chi.URLParam(r, "photoId"), 10, 64) directly. Create parseModuleIDParam similarly for moduleId.
