  Create the PostgreSQL repository implementation following the exact pattern from PostgresPatientRepository.

  **PostgresSessionRepository struct:**
  - Field: db *sql.DB
  - Constructor: NewPostgresSessionRepository(db *sql.DB) *PostgresSessionRepository

  **Methods:**

  **Create(ctx, *domain.Session) error:**
  - INSERT INTO sessions with all header fields (patient_id, clinician_id, status, scheduled_at, started_at, completed_at, patient_goal, fitzpatrick_type, is_tanned, is_pregnant, on_anticoagulants, photo_consent, notes, version, created_at, created_by, updated_at, updated_by).
  - Use RETURNING id to set the session ID on the struct.
  - Use parameterized SQL ($1, $2, ...).

  **GetByID(ctx, int64) (*domain.Session, error):**
  - SELECT all session fields by ID. Scan into domain.Session.
  - Also query session_indication_codes for the session's indication code IDs (separate query or LEFT JOIN).
  - Return wrapped ErrSessionNotFound if sql.ErrNoRows.

  **Update(ctx, *domain.Session) error:**
  - UPDATE sessions SET ... WHERE id = $1 AND version = $expected_version.
  - Increment version. If zero rows affected, return wrapped ErrSessionVersionConflict.
  - Update metadata: updated_at, updated_by.

  **UpdateStatus(ctx, id, status, expectedVersion, userID) error:**
  - UPDATE sessions SET status = $1, version = version + 1, updated_at = $2, updated_by = $3 WHERE id = $4 AND version = $5.
  - If zero rows affected, return wrapped ErrSessionVersionConflict.

  **List(ctx, SessionFilter) (*SessionListResult, error):**
  - Build dynamic WHERE clause from filter (patient_id, clinician_id, status -- each optional).
  - COUNT query for total. SELECT with LIMIT/OFFSET for pagination.
  - Follow exact pattern from PostgresPatientRepository.List.

  **ListByPatient(ctx, patientID) ([]domain.SessionSummary, error):**
  - SELECT id, created_at, status FROM sessions WHERE patient_id = $1 ORDER BY created_at DESC.
  - Scan into []domain.SessionSummary.

  **SetIndicationCodes(ctx, sessionID, codeIDs) error:**
  - DELETE FROM session_indication_codes WHERE session_id = $1.
  - INSERT new rows for each code ID. Use a loop with individual inserts (simple approach, not batch).
  - This replace-all pattern is acceptable since the junction table is small.

  Use `fmt.Errorf("...%w", err)` for error wrapping. Use `database/sql` for scanning nullable fields.
